기본 속성 선택자

기본 선택자 뒤에 [] 를 사용하여 속성과 속성값을 사용한다. (주로 id나 class 속성 아닌 것들에 적용한다.)

선택자[속성=속성값] {설정 내용} 이런 형식으로 사용된다.

div1
div2
div3
div4
test

후손 선택자와 자손 선택자

자손 선택자 : 바로 아래의 요소, 후손 선택자 : 하위 요소 전부

자손 선택자 사용 : 선택자>자손선택자 {설정내용} 의 형식
        후손 선택자 사용 : 선택자 후손선택자 {설정내용} 의 형식

자손입니다.

나도 자손입니다.

    리스트
  • 나는 ul의 자손이면서 동시에 div의 후손입니다.
  • 나도 ul의 자손이면서 동시에 div의 후손입니다.

동위 선택자

동위 관계(형제 관계) 에서 뒤에 위치한 태그를 선택할 때 사용한다.

바로 뒤의 요소 선택 시 : 선택자 a + 선택자 b {설정내용} 형식
         바로 뒤에 있는 모두를 선택할 시 : 선택자 a ~ 선택자 b {설정내용} 형식
div-test1
div-test2
div-test3
div-test4
div-test5

반응 선택자

사용자의 움직임에 따라 달라지는 선택자

사용자가 클릭한 태그 : 선택자:active{설정내용},
         사용자의 마우스가 올라가 있는 태그 : 선택자:hover {설정내용} 으로 사용된다.
active 테스트
hover 테스트

상태 선택자

입력 양식의 상태에 따라 선택되는 선택자

체크 상태의 input 태그 선택 : input태그:checked {설정내용}

초점이 맞추어진 input 태그 선택


사용 가능한 input 태그 선택 

태그:enabled {설정내용}, 태그:disabled {설정내용}
